# Machine Learning on Big Data Projects

A collection of projects and exercises from the Machine Learning on Big Data course.

## Contents

This repository includes:
- **Datasets**: 
  - `ml-100k`: MovieLens dataset with ratings, movies, and user information.
  - `data.csv`: Financial Sentiment data.
- **Notebooks**: Project implementations.

## Projects

### Recommendation System
A movie recommendation system using PySpark and the MovieLens dataset. Includes analysis of user ratings, popular movies, and user demographics, as well as the development of content-based, collaborative filtering, and hybrid recommendation systems.

**Key Topics:**
- MovieLens dataset exploration (ratings, movies, users)
- PySpark SQL for data processing
- Recommendation algorithms

### Sentiment Analysis
A multi-class financial sentiment analysis and classification using PySpark. Comparison of different feature extraction techniques (TF-IDF, Word2Vec) with various machine learning models.

**Key Topics:**
- NLP preprocessing (tokenization, stop words removal, stemming, lemmatization)
- Feature extraction: TF-IDF, TF-IDF with bigrams, Word2Vec
- Classification models: Logistic Regression, Decision Trees, Random Forest
- Model optimization and evaluation
- PySpark ML pipelines

## Getting Started

Each project is contained in a Jupyter notebook. Run the notebooks in your preferred Jupyter environment to explore the implementations.
